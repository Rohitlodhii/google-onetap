# Google One Tap Auth — Skill

This directory contains the **google-one-tap-auth** skill for [opencode](https://opencode.ai).

## Install

To use this skill in your project:

```bash
# From the root of your opencode project
opencode skill install C:\path\to\skillfolder
```

Or directly from a GitHub repository:

```bash
opencode skill install <github-username>/<repo-name>
```

## Usage

Once installed, opencode will automatically load the skill when you ask questions about Google One Tap authentication, Google Identity Services (GIS), or related integrations.

## Repository listing on skills.sh

If you own a repository listed on [skills.sh](https://skills.sh), add a `skills.sh.json` file to control how its repository page appears. See the [skills.sh.json](skills.sh.json) file in this repository for an example configuration.
